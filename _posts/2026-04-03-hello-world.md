# Hello, World!

Every developer site needs a first post, so here it is.

This site is built with [Jekyll](https://jekyllrb.com/) — a static site generator that turns Markdown files into a
fully functioning website. The whole thing is hosted for free on [GitHub Pages](https://pages.github.com/), which
means the source code and the deployed site live in the same repository.

## Why Jekyll?

A few reasons:

- **No database** — everything is a file, which makes version control trivial
- **Markdown** — writing in Markdown is fast and keeps me focused on content
- **Free hosting** — GitHub Pages deploys automatically on every push to `main`
- **Themeable** — the Cayman theme gives a clean look without any CSS wrestling

## What's next?

I plan to use this site to document projects, share notes, and occasionally write about things I'm learning.
Stay tuned.
